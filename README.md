# ParkEase Web

**Created by Bhagya**

A comprehensive parking management solution designed to connect drivers with parking spot owners.

## Features

### 🚗 For Drivers
- **Search Parking**: Find available parking spots based on location.
- **Booking Management**: Book spots in advance and manage your reservations.
- **Navigation**: Get directions to your booked spot.
- **Profile Management**: Manage your vehicle details and personal information.

### 🅿️ For Parking Owners
- **Owner Dashboard**: innovative dashboard to manage your parking spots.
- **Real-time Status**: View current occupancy and upcoming bookings.
- **QR Scanning**: seamless check-in/check-out for drivers using QR codes.
- **Earnings Tracking**: Monitor your revenue from parking bookings.

### 🛡️ For Admins
- **Admin Dashboard**: Oversee platform activity and user management.
- **System monitoring**: Keep track of overall system health and statistics.

## Tech Stack
This project is built with modern web technologies:
- **Frontend**: React, TypeScript, Vite
- **Styling**: Tailwind CSS, Shadcn UI
- **Backend/Database**: Supabase
- **Icons**: Lucide React
- **Maps**: Leaflet

## Getting Started

To run this project locally:

```sh
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd remix-of-parkspot-connect

# Step 3: Install dependencies
npm i

# Step 4: Start the development server
npm run dev
```

## Deployment
This project can be deployed to any modern web hosting service like Vercel, Netlify, or similar platforms that support Vite/React applications.
